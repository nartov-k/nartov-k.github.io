Web-app: Real-time Emotion Recognition

A web app containing a computer vision model that recognizes your emotions in real-time through a web camera. The model was crafted through transfer learning applied to ResNet-50 and assembling my own dataset. The trade-off between decent accuracy (81.94%) and the required computational load was reached by only training half of the original network structure. The app utilizes WebSocket to handle asynchronous interaction between the front and the back.
